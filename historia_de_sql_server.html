HISTORIA DE SQL SERVER

SQL Server, como motor de bases de datos relacionales que es, ha pasado por diferentes versiones. En 1989 surgió la versión 1.0 del motor. Pero las primeras versiones comerciales del mismo surgieron en 1995, con la versión 6.0, y en 1998 con la versión 7.0. Dichas versiones eran manejadas en ambiente D.O.S., es decir, con consola de comandos. 

Para el año 2000, surgió la versión SQL Server 2000. Fue la primera versión en ambiente gráfico que existió. Por eso, muchas empresas hicieron la migración hacia esta nueva versión.

En el año 2005, aparece la versión 2005 del motor. Es la primera versión que sale en su versión libre o gratuita. Esto lo hizo Microsoft debido a que en esa época MySQL estaba cogiendo mucha fuerza y debía contrarrestar esa competencia.

En el 2010, aparece la primera versión para la nube, SQL Server 2010.

Actualmente está vigente la versión SQL Server 2019. 

Es de aclarar que cada versión incorpora mejoras con respecto a la versión anterior. Muchas veces estas mejoras son internas, no tanto de interfaz gráfica.

En cuanto a las ediciones, existen cuatro (4):

  • Enterprise: Es la edición mas completa, sin restricciones. Hecha para ambientes de producción.
  • Developer: Es igual a la Enterprise, pero para ambientes de desarrollo.
  • Standard:  Tiene ciertas restricciones como la cantidad concurrente de procesadores soportados.
  • Express: Es la versión gratuita, la cual, por ejemplo, tiene un límite de hasta 10GB de almacenamiento.
